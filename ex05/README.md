## 負けるな！こうかとん(ex05/fight_koukaton.py)

- ex05/fight_koukaton.pyを実行すると、1600x900のスクリーンに草原が描画され、動く爆弾を撃墜するゲーム(にしたかった)
- 爆弾にぶつかると終了
### 操作方法
- 矢印キーでこうかとんを動かす
### 追加機能
